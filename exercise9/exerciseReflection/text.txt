Задание (Рефлексия)
1.      Создайте несколько классов, содержащих метод init(String str, int a, int b). 
Скомпилируйте их и поместите в разные jar-архивы. 
Каждый из классов делает определенные действия с заданными параметрами. 
Например, один из них умножает, другой складывает. 
В качестве результата выводится имя класса, строка str и результат работы над параметрами a и b.  
2.       Создайте основной класс, который анализирует заданную директорию и запускает метод init для каждого класса. 
Путь к директории, где расположены jar-архивы не задавать через classpath. 
Обработать ситуацию, если в классе не было найдено метода init или у него другое количество параметров. 
Выдать сообщение пользователю.
